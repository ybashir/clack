For Launch
  Test the production app at ncvgl.clack.com using the Chrome MCP browser. This is a pre-release QA pass — everything must be polished.
  Rules:
  1. Before starting, pull all open GitHub issues to avoid duplicating work. Also review closed issues for inspiration on related areas to probe (but don't re-test already-fixed bugs).
  2. Navigate the app as a real user would. Test every feature you encounter: messaging, DMs, threads, file uploads, search, profiles, navigation, emoji reactions, etc.
  3. Look for visual glitches, broken interactions, missing functionality, inconsistencies between similar features (e.g. channels vs DMs), and edge cases.
  4. When you find a bug, file a GitHub issue immediately with clear steps to reproduce, expected vs actual behavior, and severity.
  5. Keep going — explore different areas of the app, don't just test one section. Try different directions each pass.
